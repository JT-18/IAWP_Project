My Projects

Climate Classifier

The project was basic image classifier which when given an input image would classify the image into one of the 13 classes of the climate present in the world as per the Koppen climate classification. The output was input image captioned with the climate which was present in that image.The key feature of our project was it would work on images which are taken on mobile phones and Cameras.The classifier could be used in websites which have large collection unlabelled images etc.

Photo-Police

The project detects if the photo fed to the model as an input is photoshopped/manipulated or not.The output of the project is the input image with a label having the value of the amount of manipulation detected by the model in the given image.This project has numerous applications in the field of cyber-security and social media applications.Furthermore, we can extend this project to detect any manipulation in the videos which will curb the spread of fake videos and thus rumours and fake news.
